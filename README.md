# Peter Geipel's Random Password Generator

## Description

This generator is a simple, highly-effective tool to create thoroughly randomized passwords.  Through a series of prompts, a user may select a password anywhere between 8 and 128 characters.  These characters may be uppercase, lowercase, numerical, or special.  After completing the prompts, the user will automatically be provided with a password according to their specifications.
